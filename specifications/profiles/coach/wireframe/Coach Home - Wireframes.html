Notes & décisions à arbitrer

  • Création d'équipe : carte fantôme "+" dans la grille (A, C) ou bouton header (B, D). Dans tous les cas → modal wizard 2 étapes (nom + cat. → couleurs/logo). Décide pour moi = je propose carte fantôme + bouton header (les deux), comme dans la maquette A.
  • Suppression : kebab ⋮ sur chaque carte → menu (Renommer · Couleurs · Supprimer) → confirmation modale. Visible aussi en mode liste.
  • Accès équipe : clic sur la carte/ligne entière. Pas besoin de bouton "Ouvrir".
  • Limite 1–15 équipes : A et C respirent jusqu'à ~8, après quoi ça scrolle beaucoup. B et D scalent mieux pour 10+.
  • Mobile : nav bottom-tabs sur tous les modèles. Sidebar (D) devient drawer ☰.
  • Bilingue : structure prête, on bascule via i18n dans la version hi-fi.
  • Prochaine étape : tu choisis 1–2 directions, et je passe en hi-fi avec ton design system (boutons / cards / sidebar du kit).
