# Joueur — Tableau de bord

## URL

`/dashboard`

## Description

Page d'accueil après login. Vue centrée sur ses équipes et leurs matchs. Lecture seule.

## Contenu affiché

### Mes équipes

- Cards de ses équipes avec : nom, couleur, son numéro de maillot, son poste
- Lien vers le détail de chaque équipe

### Prochains matchs de mes équipes

- Liste des 5 prochains matchs des équipes dont il est joueur
- Informations : date, adversaire, lieu
- Lien vers le détail du match (lecture seule)

### Classement actuel

- Classement de ses équipes dans leur poule
- Position, points, matchs joués

### Message si aucune équipe

« Vous n'êtes encore assigné à aucune équipe. »

## Aucune action de modification

Toutes les données sont en lecture seule.
